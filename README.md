# Timelock PrivEsc Detection Agent

## Description

This agent detects privilege escalation in a governance system's timelock. 

## Supported Chains

- Ethereum

## Alerts

- TIMELOCK-PRIVESC-1
  - Fired when __

  - Type is always set to "suspicious"

- TIMELOCK-PRIVESC-PRIVESC-3

- TIMELOCK-PRIVESC-PRIVESC-4
